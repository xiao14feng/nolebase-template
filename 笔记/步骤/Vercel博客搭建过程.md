# 博客搭建过程
>图床：gitee（xiaofeng）
博客：github（fz9384838）、vercel（182、xiao12feng@outlook）
域名：阿里域名（xiao13）

1. 首先从GitHub上fork这个对应的项目，[博客模版](https://github.com/makerjackie/nolebase-template)
2. 然后从本地GitHub Desktop打开（方便操作），之后就可以修改项目里面的 笔记 目录，修改成功直接上传
3. 注册Vercel然后拉取这个项目，并且进行部署（自动化）
4. 后续的更改都不用在管，Vercel每个月能够更新资源够我折腾啦。
5. (可选)域名配置：先在vercel添加这个域名，接着拿从阿里买好的域名，除了cname对应的值修改为cname.vercel-dns.com，其他按照默认的进行配置就行。
